# Provider Credentialing SLA & Backlog Intelligence

Industry-standard healthcare analytics project using Snowflake, SQL, and Power BI

## Business Problem
Healthcare credentialing teams need clear visibility into provider onboarding volume, turnaround time, backlog aging, service level agreement (SLA) risk, and state-wise performance. Operational leaders often struggle to identify where pending files are accumulating, which analysts or states have the highest delays, and what document or verification issues are slowing network readiness.

## Objective
Build a Snowflake-based analytics solution that:
1. loads raw provider credentialing records
2. transforms the data into reporting-friendly structures
3. computes operational KPI metrics in SQL
4. feeds Power BI dashboards for executive review

## Tech Stack
- Snowflake
- SQL
- Power BI
- CSV / Excel
- GitHub

## Project Architecture
Raw CSV -> Snowflake staging table -> curated reporting view -> KPI SQL -> Power BI dashboards

## Dashboard Pages
1. Executive Overview
2. Backlog & Aging
3. Quality & Exceptions
4. Productivity & Throughput
